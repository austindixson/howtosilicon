Thesis

Long-horizon local agents do not fail because 35B models are dumb. They fail because the live prompt is treated as durable memory on hardware that cannot host cloud-sized KV. Disk is the system of record. Context is a cache. Completion is a steer — not a tool-budget tombstone.
