Same model · context engine OFF

Matched A/B arm: identical Qwen 35B weights and tool seed, with context memory disabled. Raw tool bulk stays inline. Useful only to measure what the engine buys you.

Measured (matched seed)

  • Wall: ~129 s
  • Externalized: 0
  • Fitted est: ~24k (no hard CM ceiling on dispatch assembly)
  • Ratio vs ON: 5.12× tokens · 5.7× wall

What fails on long tasks

  • Higher TTFT and decode cost every turn
  • Closer to Metal / KV cliff on multi-hour loops
  • No evidence retrieve path — history is the only memory
